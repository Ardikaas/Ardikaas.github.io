## 🛠 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/Ardikaas/Ardikaas.github.io
   ```
2. Navigate to the project directory:
   ```bash
   cd Ardikaas.github.io
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
